How to use this?

So you will need secret token from notion and link to your database.

Guide to get secret token

Check this tutorial or some other to generate your integration key link

Guide to get link to your database

Click on "Click to open menu"

Then click on "Copy link"

And yeah, you have your database link that you can paste in home page

Database format

Database should have 5 required columns to have proper usage:

  • done
  • name
  • tags
  • link
  • additional comment
